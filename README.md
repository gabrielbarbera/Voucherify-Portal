# Voucherify Portal

A React-based portal for managing product vouchers and redemptions.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npm run dev
   ```

The application will be available at `http://localhost:5173`
